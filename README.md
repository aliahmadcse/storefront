# Store Front

## Project Setup

### Requirement

- Python 3.9
- Pip 21.1

### Setting Up

- Run `pip install pipenv`
- Run `pipenv install --ignore-pipfile`

### Run Server

- Open a terminal inside the virtual environment created by pipenv using
  `pipenv shell`
- Run `python manage.py runserver` to start server at `PORT 8000`
